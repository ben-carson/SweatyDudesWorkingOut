# Product Overview

FitTrack is a social fitness tracking application that combines workout logging with community features. Users can track their exercise sessions, participate in fitness challenges, and connect with friends to share progress and achievements.

## Core Features
- **Workout Tracking**: Log exercises, sets, reps, weights, and duration
- **Social Challenges**: Create and participate in fitness challenges with friends
- **Progress Monitoring**: View workout history, stats, and personal records
- **Social Feed**: Share workouts and interact with friends' fitness activities
- **Real-time Sessions**: Active workout tracking with cross-tab synchronization

## Target Users
Fitness enthusiasts who want to track their workouts while staying motivated through social interaction and friendly competition.

## Key User Flows
1. Start workout → Log exercises → Complete session
2. Create/join challenges → Log entries → View leaderboards
3. Browse friend activity → Like/comment → Share own workouts
4. View progress charts → Analyze trends → Set new goals
